Instagram Reels Pipeline

릴스 데이터를 정리하고 분석해서 다음 기획 초안까지 이어가는 워크스페이스

정보 레퍼런스와 형식 레퍼런스를 나눠 저장하고, 처리 결과와 분석 산출물을 파일 기준으로 관리합니다.

Reference Intake

릴스 레퍼런스 등록

링크를 한 줄에 하나씩 넣으면 저장됩니다. 제목, 토픽, 태그는 필요한 만큼만 보정하면 됩니다.

세션 확인 중

Instagram 로그인 세션이 없으면 저장만 되고 자동 처리 단계는 시작되지 않습니다.

Information

정보 레퍼런스

핵심 정보, 주장, 인사이트를 정리해두는 레퍼런스입니다.

Format

형식 레퍼런스

구성, 전달 방식, 화면 리듬을 참고하는 레퍼런스입니다.

Unassigned

미분류 레퍼런스

아직 정보/형식으로 나누지 않은 항목입니다.

Planner

기획안 생성

선별한 정보 레퍼런스와 형식 레퍼런스를 묶어 초안을 만듭니다.

정보 레퍼런스 선택

형식 레퍼런스 선택

Saved Plans

저장된 기획안

생성한 초안을 다시 열어 수정하거나 상태를 바꿉니다.

Queue

작업 큐

백그라운드 작업은 순서대로 처리되고, 서버를 재시작하면 큐 기록만 초기화됩니다.
